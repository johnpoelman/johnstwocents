Streamdeck Installation on Linux

Nov 11, 2022
3 min read
linux streaming

Nowadays streaming is a widely practiced hobby. Gaming, e-sports, etc… are all things that aid to the popularity of streaming as a whole. A lot of people dream about streaming content for a living and for that you need to have a setup that enables you to do a lot of different stuff on stream, there comes the Streamdeck into play. The Streamdeck is actually a buttonbox that you can program with all kinds of functionality to enhance your streams or even you worklife.

Like with a lot of peripherals they are built for using Windows and really lack support for the linux OS. Thanks to the upcoming popularity of linux gaming the community made it possible to use the device on linux as well.

Gaming on linux is getting more and more popular you say? Of course, thanks to, for an example, the Steamdeck. This nifty portable console Valve has brought to life is running completely on the linux OS. You should definately check it out if you’re interested in this device.

Installation

The Streamdeck is a peripheral that communicates with a software running on your computer to see which action it has to do when pressing some kind of button. The software that we are installing today is a native implementation of the Streamdeck application created by Timothy Crosley.

The first thing to do is installing the dependencies for the streamdeck-ui package, using the command below.

sudo apt install python3-pip libhidapi-libusb0 libxcb-xinerama0

After this we need to make sure that we set up special permissions to use the application with a regular system user. Use the code below to create the streamdeck rules.

sudo nano /etc/udev/rules.d/70-streamdeck.rules

Once the file is created paste the following rules in it. One thing to mention, if you have the Streamdeck MK2 don’t forget to paste the last rule, for the MK1 that one is not necessary. It never hurts to paste them all, just in case.

SUBSYSTEM=="usb", ATTRS{idVendor}=="0fd9", ATTRS{idProduct}=="0060", TAG+="uaccess"
SUBSYSTEM=="usb", ATTRS{idVendor}=="0fd9", ATTRS{idProduct}=="0063", TAG+="uaccess"
SUBSYSTEM=="usb", ATTRS{idVendor}=="0fd9", ATTRS{idProduct}=="006c", TAG+="uaccess"
SUBSYSTEM=="usb", ATTRS{idVendor}=="0fd9", ATTRS{idProduct}=="006d", TAG+="uaccess"
SUBSYSTEM=="usb", ATTRS{idVendor}=="0fd9", ATTRS{idProduct}=="0080", TAG+="uaccess"

When that is done you can save the file and reload the rules by rebooting or using the command below.

sudo udevadm control --reload-rules

The last step is to use Python to install the Streamdeck application by using the command below.

pip3 install --user streamdeck_ui

Now the only that’s left is making the installation work by having the correct path in your profile. Depending on which shell you are using this will differ but if you are using BASH like me you should find a .bashrc file in which your path is defined. just add the following line to the bottom of that file if $HOME/.local/bin isn’t already a part of your path.

PATH=$PATH:$HOME/.local/bin

Running the program

You can run the program from your terminal in the background with the command

streamdeck &

Personally I added this command to my startup applications for ease of use. That way I can always open up the software to make a change. It’s up to you wether to decide this is a good idea or not. In any case, by now you have a working installation of your Streamdeck software and you’re about to do your first linux stream!

Game on and see you later :)
